This template consists of several files and directories. Many files have to go in special places. In the following

⟨SW⟩
is your instalation folder. For me it is "C:\progfiles\swp55"
⟨HOME⟩
is your home directory. For me it is "C:\Users\theo\Documents".

(0). Download style files etc

  • mun-thesis.cst goes in ⟨SW⟩\styles\report
  • The following all go in a new directory ⟨SW⟩\TCITeX\TeX\LaTeX\munthesis
    • mun-thesis.sty
    • named.cite.sty
    • code.sty
    • comment.sty
    • thesis.zeros.sty
  • thesis.bib goes in ⟨SW⟩\TCITeX\BibTeX\bib\munthesis\
  • The following files all go in folder ⟨SW⟩\Frags\
    • algorithm.frg
    • figure.frg
    • listing (nonfloating).frg
    • listing (floating).frg
    • listing (from file).frg
    • listing (inline).frg
    • parcode.frg
    • shortcite.frg
    • singlespaced.frg
    • sscode.frg
    • ssfigure.frg
    • sstable.frg
  • Unzip directory listings.zip so that it is under ⟨SW⟩\TCITeX\TeX\LaTeX\contrib\

(1) Next, download file munthesis.zip and unzip it so that munthesis is a subdirectory of ⟨HOME⟩.

(2) With Scientific Word (or Workplace), open Master.tex in ⟨HOME⟩\munthesis.

  • Compile (Typeset >> Compile...), check "Generate a bibliography", and click Ok.
  • Preview (Typeset >> Preview...), click Yes.
  • Read it.

(3) If you had no problem with step (2), make a copy of ⟨HOME⟩\munthesis called ⟨HOME⟩\mythesis (or whatever you want to call it). Modify to make your thesis. I suggest you keep ⟨HOME⟩\munthesis untouched as documentation and so you can see how I did various things. Add bibliographic references to thesis.bib. (See BibTeXing for documentation.)

(4) Remember to back up your thesis frequently. A good way to do that is to put the \munthesis directory in a dropbox.com folder.
